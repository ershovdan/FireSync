# Fire Sync
Fire Sync is a server side of MF file syncer system. It consists from 
the core (created with Java), web server (Python, JS) for handling if file requests 
and user interface and database (PostgreSQL).
---








### List table statuses
- 0 - disabled/hidden
- 1 - active
- 2 - paused
- 4 - zipping
- 5 - just created
